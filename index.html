HOW TO SET UP SHEETS SYNC

1. Open your Google Sheet
2. Copy the ID from the URL (the long string between /d/ and /edit)
3. Create a Google Cloud project, enable Sheets API, create an API key
4. Paste both above and save

The app writes only raw input columns (C–G):
C: Date · D: Where · E: Odometer · F: Gallons · G: Total Paid

Columns A (Year), B (Year-Month), and H+ formula columns are left untouched.
